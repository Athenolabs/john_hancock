#### How to install


1) Install signaturit_sdk package
   
    a) pip install signaturit_sdk

2) Get John Hancock

	a) bench get-app john_hancock https://github.com/ratskin/john_hancock.git

3) Install John Hancock

	a) bench install-app john_hancock
